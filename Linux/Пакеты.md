Внутренняя база:

 dpkg 
~ dpkg -l (л мал)---> все установленные пакеты
~ dpkg -s telegram-desctop ---> инфа о приложении
~ dpkg -L telegram-desctop ---> какие файлы приложения есть в системе
~ dpkg -i program.deb ---> установить скачанный пакет
~ dpkg -r program.deb--->
